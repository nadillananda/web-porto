# web-porto

Nadilla's dummy website portfolio. A minimalist, cinematic portfolio homepage built with Next.js, React, Tailwind CSS, and TypeScript.

## Features

- Full-screen hero section with cinematic background
- Large, bold typography with split headlines
- Smooth fade-in animations
- Responsive design for all screen sizes
- Minimal aesthetic with wide spacing
- Navigation bar with thin fonts
- Footer with poetic tagline and social links
- About, Journal, and Contact sections
- Scroll-to-top functionality

## Getting Started

First, install the dependencies:

```bash
npm install
```

Then, run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Tech Stack

- **Next.js 14** - React framework
- **TypeScript** - Type safety
- **Tailwind CSS** - Styling
- **React** - UI library
